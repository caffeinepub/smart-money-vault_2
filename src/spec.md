# Specification

## Summary
**Goal:** Remove all support-related wording from the Settings → Subscription tab UI.

**Planned changes:**
- Remove or filter out the feature strings “Community support”, “Priority Support”, and “Dedicated Support” so they do not render anywhere in the Subscription tab (including tier feature lists).
- Ensure Subscription tier cards continue to render normally with all other feature entries unchanged and the subscribe flow remains unaffected.

**User-visible outcome:** When viewing Settings → Subscription, users will no longer see “Community support”, “Priority Support”, or “Dedicated Support” anywhere in the UI.
